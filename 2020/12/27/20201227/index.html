2020年12月27日

  • Diary

这是第一篇在移动设备上创建，完成并上传的日记博客，算是Milestone了。

操作步骤：

  • 在iPad Pro 11寸一代及以上的iPad安装iSH Shell。(A12X,A12Z和A14的平板都应该是可以的)
  • 依次从apk安装：
    git
    openssh
    node
    npm
  • 如果是现有博客移植，直接把hexo博客目录（带node_modules的）直接发到iPad的iSH文件系统内的home目录里。
  • git config —global user-name和user-email。
  • ssh-keygen然后ssh-add。
  • 把key拷到Github下的SSH Keys。
  • ln -s modules里的hexo-cli hexo博客目录下/hexo
  • ./hexo g
  • ./hexo d
  • 完成！

P.S. 我在地铁上用iPad Pro写的这段。
