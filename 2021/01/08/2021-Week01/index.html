网站加入Categories对网页正确分类

创建Git Server - Gitea

hexo后期页面的开发预览仍然离不开电脑，所以准备建立一个Git服务器来同步框架代码和文章的Markdown。

最早看到一个基于IIS的.NET网页端Git服务器，虽然我现在的核心服务器跑的是Windows Server 2016 Datacenter，但是我IIS就不想托管应用程序的网页，第一是这台服务器暴露公网，怕一个0-Day把你的服务器给搞瘫，第二是自己不能保证一个第三方的Git服务器满足你所有要求。最后无意间看到了Gitea，Go语言，nodejs，多平台，Windows上deploy一个exe文件就可以了，很方便。

当然你还需要一个SQL Server - MySQL Server 8

用于存放用户信息的数据库，未来还能用来干很多事的。
