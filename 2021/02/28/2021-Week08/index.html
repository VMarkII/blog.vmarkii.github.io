家中网络正式升级到了1000Mbps并外加海外精品网

2990元外加50元每月的海外精品网，Github，微软Auth和iTunes下载的速度都有很大提升。

买了新的6SATA盘位的ITX机箱

机箱名字叫蜜罐超存，里面配的200W电源换成了我之前ITX机箱内的350W 1U电源。200W根本带不动i7 8700和B365M。
之前在写完上周周记考虑把SAS硬盘全部搬入这个新机箱，然后发现我要是插入了整列卡，机箱就关不上了，就压根没有给我开PCIe机箱外槽，又全部装回去了。

Windows Server 2019 Datacenter还在摸索

这一周我因为AD DS配置问题，光是重装Windows Server 2019就重装了4次，先暂时不这么做了。

直到今天，我才知道，原来I211网卡是可以刷成I210网卡从而实现Windows Server免驱动（之前都要禁用驱动数字签名然后安装加过VID的驱动），可见我的知识是多么匮乏了。

而且现在的链路聚合都不是在设备管理器里加组了，你只要在服务器管理器里设置网络组就可以了，至少有一些进展。（2.0Gbps至少对多个Wi-Fi 6设备同时访问没什么问题了）

工作中获得Visual Studio Professional订阅

这个东西，和Microsoft Azure，Adobe Creative Cloud都有一个共同点：

如果你不是专业的个人开发者或者内容创造者，你个人可能一年都用不上10次，肯定是不舍得买这个了。能在工作中获得企业正版授权并且学习知识，这些实际操作都是书上看不到的。

这样的好机会，怎么能错过呢。
