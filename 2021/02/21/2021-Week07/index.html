Windows Server Backup的小问题

Windows Server备份的UI，wbadmin.msc，会出现打不开的情况。

解决方法：以管理员身份运行以下命令：

购买了Windows Server 2019 Datacenter和新的机箱

准备进行服务器升级。

基本计划：

J3455B-ITX主板+一个NAS系统（正在Windows Server 2016，FreeNAS和UnRAID中选），装上现在所有的SAS盘和整列卡，使用旧机箱。

B365M-ITX主板+一个Hypervisor（Proxmox VE内安装Windows Server 2019或者Windows Server 2019使用Hyper-V），将SAS盘和整列卡移除，使用新机箱。
